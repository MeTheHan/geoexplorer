🌍 GeoQuest 2025

Modern, interaktif dünya ülkeleri bulmaca oyunu

🚀 Nasıl Çalıştırılır?

  1. Python'un yüklü olduğundan emin olun
  2. Pygame kütüphanesini yükleyin: pip install pygame
  3. geoquest_2025.py dosyasını indirin
  4. Terminal/CMD'den: python geoquest_2025.py
📥 Oyunu İndir

  • 🌐 Çoklu dil desteği (Türkçe/İngilizce)
  • 🎯 Modern 2025 arayüz tasarımı
  • 🕹️ İnteraktif harita
  • 📱 Mobil uyumlu
  • ⚡ Gerçek zamanlı skor takibi
